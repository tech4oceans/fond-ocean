# Презентация: Фонд «Океан» / Ocean Fund

## Цель

Кратко представить фонд ученым, волонтерам, разработчикам, музеям, университетам и потенциальным партнерам.

## Структура

1. Название: Фонд «Океан» / Ocean Fund.
2. Миссия: открытая исследовательская, образовательная и технологическая инфраструктура для защиты океана.
3. Почему океан важен: климат, биоразнообразие, общество, данные, международное сотрудничество.
4. Направления: research, data, education, technology, partnerships.
5. GitHub-штаб: документы, данные, задачи, templates, roadmap.
6. Первые исследовательские темы: biodiversity, climate, pollution, data infrastructure, blue economy.
7. Как присоединиться: issue, pull request, партнерский запрос, событие.
8. Следующие шаги: проверка материалов, первый release, GitHub Pages, партнерские письма.

## Что подготовить перед показом

- актуальные контакты фонда;
- подтвержденный логотип или визуальный стиль;
- 2-3 проверенных примера данных;
- список материалов, которые можно публично распространять.
