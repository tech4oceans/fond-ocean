# Exhibition Booth Outline

## Цель

Подготовить структуру стенда фонда для выставки, музея или научного фестиваля.

## Зоны стенда

| Зона | Содержание |
| --- | --- |
| Океан и жизнь | Короткие факты о роли океана для климата и биоразнообразия |
| Данные океана | Карты, открытые источники, примеры переменных |
| Исследовательский вопрос | Один понятный вопрос, который посетитель может изучить |
| Как участвовать | Волонтерство, data tasks, образовательные материалы |
| Партнерство | Форматы для университетов, музеев и научных центров |

## Материалы

- QR-код на GitHub-репозиторий;
- короткая презентация фонда;
- карточки источников данных;
- визуализация батиметрии или биоразнообразия;
- форма интереса без публикации персональных данных в репозитории.
