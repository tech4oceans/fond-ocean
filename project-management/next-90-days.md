# План на 90 дней

## Цель

Перевести репозиторий из базовой структуры в работающий публичный штаб фонда: с задачами, обсуждениями, данными, презентациями и первыми партнерскими процессами.

## Месяц 1

- отдельный GitHub-контур фонда;
- безопасная публичная структура;
- первые issues и discussions;
- первичный реестр данных;
- первые outreach templates.

## Месяц 2

- демонстрационный notebook на открытых данных;
- GitHub Pages с навигацией по миссии, исследованиям и данным;
- список целевых организаций;
- первая версия partner deck;
- карта конференций и выставок.

## Месяц 3

- первый публичный release;
- 2-3 проверенных research briefs;
- 1-2 визуализации на открытых данных;
- регулярный backlog review;
- подготовка партнерских встреч и заявок на события.

## Метрики процесса

| Метрика | Целевое состояние |
| --- | --- |
| Открытые источники данных | 10+ проверенных карточек |
| Research briefs | 2+ черновика с источниками |
| Issues | 15+ структурированных задач |
| Outreach list | 30+ организаций без персональных данных |
| Presentations | 2 проверенных outlines |
