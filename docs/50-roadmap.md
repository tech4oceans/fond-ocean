# Дорожная карта

## Этап 1. Инвентаризация

- Сформировать индекс Bitrix24 по группе ОКЕАН.
- Зафиксировать публичную карточку фонда.
- Провести первичный аудит сайта.
- Создать GitHub-репозиторий как штаб проекта.

## Этап 2. Публичная упаковка

- Подготовить короткое описание фонда на 500, 1500 и 3000 знаков.
- Собрать страницу «О фонде».
- Подготовить страницу Дмитриенко Д.В. как президента фонда.
- Описать направления фонда и проекты.
- Создать страницу пожертвований с юридически корректными формулировками.

## Этап 3. Новый сайт

- Снять текущий WordPress-аудит.
- Удалить шаблонный и спамный контент.
- Сделать новую информационную архитектуру.
- Подготовить дизайн и контентную матрицу.
- Реализовать публичную версию сайта.

## Этап 4. Управление

- Связать Bitrix-задачи с GitHub issues.
- Вести изменения через pull request или короткие коммиты.
- Раз в неделю обновлять инвентаризацию.
- Перевыпустить Bitrix webhook и хранить доступ вне репозитория.
