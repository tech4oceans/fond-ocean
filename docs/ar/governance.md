# إدارة

![governance preview](../../assets/docs-previews/ar/governance.svg)


يصف هذا المستند كيفية احتفاظ المؤسسة بمستودع مفتوح وقبول التغييرات.

## الأدوار

| دور | مسؤولية |
| --- | --- |
| المشرفون | التحقق من الهيكل والسلامة والنغمة والامتثال لمهمة المؤسسة |
| المساهمين في البحوث | يقترح أسئلة البحث والمراجعات والمصادر |
| المساهمين في البيانات | أضف مصادر البيانات وأوصاف مجموعة البيانات ودفاتر الملاحظات |
| المساهمين في التوعية | إعداد المواد للشراكات والأحداث والاتصالات |
| المراجعين | التحقق من الحقائق والمراجع والتراخيص والملاءمة العامة |

## كيف يتم قبول التغييرات

1. يمكن اقتراح تعديلات طفيفة عبر طلب السحب.
2. تتم مناقشة الاتجاهات الجديدة والشراكات والإعلانات العامة لأول مرة في هذا الموضوع.
3. المواد التي تحتوي على حقائق لم يتم التحقق منها تحصل على الحالة `needs verification`.
4. لن يتم قبول المواد التي تنطوي على خطر البيانات الشخصية حتى يتم التحقق منها بشكل منفصل.

## معايير الاستعداد العام

- يشير النص فقط إلى مؤسسة المحيط؛
- لا توجد اتصالات خاصة ورموز وتفاصيل مالية ومستندات شخصية؛
- تم ذكر مصادر البيانات والتأكيدات الخارجية بشكل صريح؛
- النبرة احترافية وهادئة ومفهومة دوليًا؛
- ولا توجد وعود بشأن نتائج غير موجودة.
