# 海洋噪音是看不见的污染

![ocean-noise-as-hidden-pollution preview](../../../assets/article-previews/zh/ocean-noise-as-hidden-pollution.svg)

当谈到海洋污染时，人们首先想到的是塑料、石油、污水或化学品。但海洋附近还有另一种污染，在照片中很难看到，因此更容易被低估。这是噪音。

对于人类来说，海洋似乎是一个广阔而安静的环境，但对于许多海洋生物来说，声音是感知世界最重要的渠道之一。动物通过声音导航、寻找伙伴、交流、寻找食物和识别危险。因此，人为噪声的增加不仅改变了“背景”，而且改变了海洋生物生存的条件。

噪声源有很多：航运、建筑、地震勘测、军事活动、工业基础设施。它们的影响范围从短期压力到行为和迁徙的长期干扰。特别敏感的是声环境起关键作用的物种。

海洋噪音问题也很重要，因为它与通常的生态直觉不太相符。塑料可以显示在你的手中。可以拍摄浮油。声污染需要不同的语言：图表、水听器数据集、解释器、航运地图和患者科学交流。

这就是为什么噪音话题很好地说明了为什么社会需要开放数据和高质量的科学翻译。如果没有它们，对话很容易就会变成完全忽视问题，或者变成严厉但缺乏证据的陈述。与此同时，噪音是一个真正的海洋因素，需要监测、政策和公众理解。

对于海洋基金来说，海洋噪音这个话题很有趣，它是“看不见的海洋”的一个例子——这些过程在生态上很重要，但在大众的想象中却很难体现。处理这些话题尤其有价值：它们扩大了公众对海洋的了解，并表明海洋的脆弱性可能并不总是在图片中看起来很引人注目，但这并没有减轻它的严重性。
