# 漂浮城市需要指标，而不是神话

![preview](../../../assets/article-previews/zh/floating-cities-need-metrics-not-myths.svg)

关于漂浮城市的讨论，常常在热情与嘲讽之间摆动。有人把它当作气候压力下的答案，也有人把它视为昂贵幻想。但如果这一主题真的与未来海岸有关，它就需要测量体系，而不是口号。

任何漂浮系统都不应只靠漂亮渲染图来比较，而应依据清晰指标：能源、淡水、废弃物、可维护性、材料、安全、与陆地连接、生态影响、运行成本和公共价值。没有这些标准，项目仍然只是神话，而不是基础设施。

这对 Ocean Fund 尤其重要，因为基金已经处在气候、海洋、技术、水上城市、传感器和公共指数的交叉点上。因此，漂浮城市必须进入研究与评级层，而不只是进入视觉叙事。
