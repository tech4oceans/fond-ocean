## Что не переносим в GitHub

- доступы, пароли, токены, вебхуки;
- личные документы без отдельного решения;
- полные выгрузки чатов;
- документы с персональными данными, если они не являются публичной биографической справкой.

## Что можно переносить

- публичные реквизиты юрлица;
- выдержки из сайта и публичных источников;
- карту материалов Bitrix;
- дорожные карты, аудиты, технические задания.
