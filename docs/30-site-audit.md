# Аудит сайта tech4oceans.ru

Проверено: 2026-05-04.

## Общее состояние

Сайт `tech4oceans.ru` работает на WordPress и уже содержит базовый контур OceanFund: главная, о нас, контакты, статьи, индексы, рейтинги, кнопка пожертвования.

При этом текущая версия выглядит как черновая сборка: есть шаблонные фрагменты, слабая юридическая верификация, нет ясной архитектуры пожертвований и публичной отчетности.

## Сильные стороны

- Есть домен и работающий сайт.
- Есть базовое позиционирование: океан, Net Zero, полярная устойчивость, плавучие города, инновации.
- Есть контентная база по индексам: SDI, RTOHI, WEMI, CCHPI, 3DPESI, SWQI, NLWI, UCPWRI.
- Есть русско-английские материалы, пригодные для дальнейшей упаковки.

## Проблемы

- На странице `Home` остался шаблонный текст про fencing solutions.
- Есть стандартная WordPress `Sample Page`.
- Комментарии к постам не модерируются; виден массовый SEO-спам.
- В футере указан `TECH4OCEANS.COM.RU 2024`, хотя используется `tech4oceans.ru`.
- Раздел «Пожертвовать» видим в навигации, но нет понятной страницы с реквизитами, офертой, назначением платежей и отчетностью.
- Нет отдельной страницы с юридической карточкой фонда.
- Нет страницы команды/попечительского или экспертного совета.
- Нет публичного списка проектов с уровнем готовности: идея, пилот, партнерство, реализация.
- Нет аккуратной политики обработки персональных данных и согласия для форм.

## Рекомендованная структура сайта

1. Главная: кто мы, что делаем, 3-5 главных направлений, CTA.
2. О фонде: юркарточка, миссия, президент, команда, документы.
3. Направления: Арктика, морские технологии, экология, индексы, цифровые платформы.
4. Проекты: карточки инициатив с понятным статусом.
5. Индексы и исследования: отдельная витрина аналитики.
6. Партнерство: для регионов, вузов, бизнеса, международных организаций.
7. Пожертвования: реквизиты, цели сборов, отчетность, юридическая рамка.
8. Новости/публикации: модерируемый блог без комментариев или с премодерацией.
9. Контакты: форма, email, адрес, политика обработки данных.

## Приоритетные действия

- Убрать шаблонные страницы и тексты.
- Закрыть или модерировать комментарии.
- Добавить юридическую карточку и подтверждающие документы.
- Создать страницу пожертвований.
- Собрать из Bitrix публично пригодные тексты: миссия, направления, индексы, письма, презентационные тезисы.
- Переписать главную страницу в спокойном институциональном стиле.
