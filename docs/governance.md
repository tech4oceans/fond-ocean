# Управление / Governance

Этот документ описывает, как фонд ведет открытый репозиторий и принимает изменения.

## Роли

| Роль | Ответственность |
| --- | --- |
| Maintainers | Проверяют структуру, безопасность, тон и соответствие миссии фонда |
| Research contributors | Предлагают исследовательские вопросы, обзоры и источники |
| Data contributors | Добавляют источники данных, описания датасетов и notebooks |
| Outreach contributors | Готовят материалы для партнерств, событий и коммуникаций |
| Reviewers | Проверяют факты, ссылки, лицензии и публичную пригодность |

## Как принимаются изменения

1. Малые правки можно предлагать через pull request.
2. Новые направления, партнерства и публичные заявления сначала обсуждаются в issue.
3. Материалы с непроверенными фактами получают статус `needs verification`.
4. Материалы с риском персональных данных не принимаются до отдельной проверки.

## Критерии публичной готовности

- текст относится только к фонду «Океан»;
- нет приватных контактов, токенов, финансовых реквизитов и персональных документов;
- источники данных и внешние утверждения указаны явно;
- тон профессиональный, спокойный и международно понятный;
- нет обещаний о несуществующих результатах.
