# Фонд Океан

Рабочий репозиторий для инвентаризации, упаковки и развития публичной цифровой витрины Фонда развития компетенций и технологий в области морской деятельности «Океан».

## Назначение

- собрать проверяемую базу знаний по фонду;
- отделить публичные материалы от внутренних и чувствительных данных;
- вести задачи по сайту, документам, Bitrix24 и GitHub;
- подготовить структуру для нового сайта, разделов маркетинга и пожертвований.

## Быстрый старт

```powershell
$env:B24_WEBHOOK_URL = "https://<portal>/rest/<user>/<webhook>/"
pwsh -File scripts/collect-bitrix-ocean.ps1
```

Скрипт использует вебхук только из переменной окружения и не сохраняет его в файлы.

## Структура

- `docs/` — отчеты, аудиты, дорожные карты.
- `data/bitrix/` — санитизированные индексы Bitrix24.
- `scripts/` — утилиты сбора и обновления данных.

## Правила безопасности

- Не публиковать доступы, пароли, токены, персональные документы и сырые чаты.
- Публичные биографии и реквизиты проверять по открытым источникам.
- Все спорные или непроверенные утверждения помечать как требующие верификации.
