# 民众

该目录是面向公众的 GitHub Pages 或海洋基金网站展示的暂存区。

## 强制公共层

该项目的任何公共版本都需要以下两个资产：

- [对于合作伙伴](partners.md) as the institutional entry route;
- [公共任务副本](mission-copy.md) as the approved reusable description set.

每当项目用于外展时，都应提供以下支持资产：

- [合作伙伴单页机](partner-one-pager.md) as the compact first-contact brief.
- [会议/展览单页机](conference-exhibition-one-pager.md) as the compact event-facing brief.
- [活动应用包](event-application-pack.md) as the reusable submission and organizer pack.
- [索引和出版物单页](indexes-and-publications-one-pager.md) as the knowledge-system brief.

## 工作规则

此处仅保留可供公众使用的材料。请勿在此文件夹中放置私人联系人、私人文档、内部注释、凭据或未经验证的声明。

## 预期的演变

一旦项目批准了初始公共版本，该目录就应该成为第一个简单公共登陆页面的基础。
