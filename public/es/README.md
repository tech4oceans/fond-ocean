# Público

Este directorio es el área de preparación para las páginas de GitHub de cara al público o la presentación del sitio web de Ocean Fund.

## Capas públicas obligatorias

Se requieren los dos activos siguientes en cualquier versión pública del proyecto:

- [Para socios](partners.md) como vía de entrada institucional;
- [Copia de la misión pública](mission-copy.md) as the approved reusable description set.

El siguiente activo de apoyo debe estar presente siempre que el proyecto se utilice para actividades de divulgación:

- [Socio de una página](partner-one-pager.md) como escrito compacto de primer contacto.
- [Conferencia / Exposición de una página](conference-exhibition-one-pager.md) as the compact event-facing brief.
- [Paquete de solicitud de evento] (event-application-pack.md) como paquete reutilizable de envío y organizador.
- [Índices y publicaciones de una página](indexes-and-publications-one-pager.md) as the knowledge-system brief.
- [Mapa de la red oceanica](ocean-network-map.md) como dossier de orientacion hacia el ecosistema.

## regla de trabajo

Mantenga aquí sólo materiales listos para el público. No coloque contactos privados, documentos privados, notas internas, credenciales o reclamos no verificados en esta carpeta.

## Evolución esperada

Este directorio debería convertirse en la base para una primera página de destino pública sencilla una vez que el proyecto apruebe la versión pública inicial.
