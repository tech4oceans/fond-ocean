# عام

هذا الدليل هو منطقة التدريج لصفحات GitHub المواجهة للعامة أو عرض موقع الويب الخاص بـ Ocean Fund.

## الطبقات العامة الإلزامية

الأصولان التاليان مطلوبان في أي إصدار عام للمشروع:

- [للشركاء](partners.md) as the institutional entry route;
- [نسخة المهمة العامة](mission-copy.md) as the approved reusable description set.

يجب أن تكون الأصول الداعمة التالية موجودة عندما يتم استخدام المشروع للتواصل:

- [شريك صفحة واحدة](partner-one-pager.md) as the compact first-contact brief.
- [مؤتمر / معرض صفحة واحدة](conference-exhibition-one-pager.md) as the compact event-facing brief.
- [حزمة تطبيق الحدث](event-application-pack.md) as the reusable submission and organizer pack.
- [الفهارس والمنشورات صفحة واحدة](indexes-and-publications-one-pager.md) as the knowledge-system brief.
- [خريطة الشبكة المحيطية](ocean-network-map.md) كملف توجيهي نحو المنظومة.

## قاعدة العمل

احتفظ فقط بالمواد الجاهزة للعامة هنا. لا تضع جهات اتصال خاصة أو مستندات خاصة أو ملاحظات داخلية أو بيانات اعتماد أو مطالبات لم يتم التحقق منها في هذا المجلد.
