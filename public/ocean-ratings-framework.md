# Ocean Ratings Framework

![preview](../assets/public-previews/ocean-ratings-framework.svg)

This page sets the rule for how Ocean Fund will build ratings from its index layer.

## Core Rule

Ocean Fund does not publish a substantive ranking just because a number is available. A rating is publishable only when it includes:

- a named index basis;
- a clear explanation of what is being compared;
- transparent limits and uncertainties;
- public-safe source notes;
- an update rule.

## What Ratings Can Grow From This Layer

- water and monitoring ratings;
- coastal climate and resilience ratings;
- floating-city and ocean-urbanism ratings;
- public ocean intelligence and infrastructure ratings.

## Current Operational Status

The index portfolio is confirmed. The ratings framework is now public. Substantive comparative rankings will be added as each index line receives a verified methodology note.
