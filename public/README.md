# Public

This directory is the staging area for the public-facing GitHub Pages or website showcase of Ocean Fund.

## Mandatory Public Layers

The following two assets are required in any public version of the project:

- [For Partners](partners.md) as the institutional entry route;
- [Public Mission Copy](mission-copy.md) as the approved reusable description set.

The following supporting asset should be present whenever the project is used for outreach:

- [Partner One-Pager](partner-one-pager.md) as the compact first-contact brief.
- [Conference / Exhibition One-Pager](conference-exhibition-one-pager.md) as the compact event-facing brief.
- [Event Application Pack](event-application-pack.md) as the reusable submission and organizer pack.
- [Indexes and Publications One-Pager](indexes-and-publications-one-pager.md) as the knowledge-system brief.
- [Ocean Network Map](ocean-network-map.md) as the ecosystem-facing orientation brief.
