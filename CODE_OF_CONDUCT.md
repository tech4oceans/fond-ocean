# Code of Conduct

Фонд «Океан» стремится поддерживать уважительное, профессиональное и международно понятное пространство для ученых, разработчиков, волонтеров, образовательных организаций и партнеров.

## Ожидаемое поведение

- уважительно относиться к участникам и их времени;
- аргументировать предложения источниками и проверяемыми данными;
- отделять факты от мнений и гипотез;
- учитывать межкультурный контекст;
- помогать новым участникам разобраться в структуре проекта.

## Недопустимое поведение

- дискриминация, угрозы, оскорбления и преследование;
- публикация персональных данных без согласия;
- публикация токенов, паролей, приватных документов или финансовых реквизитов;
- выдача неподтвержденных договоренностей за официальные партнерства;
- использование репозитория для нерелевантной рекламы.

## Реакция

Maintainers могут редактировать, закрывать или удалять материалы, которые нарушают эти правила, а также ограничивать участие пользователей при повторных нарушениях.

Публичный контакт для вопросов будет добавлен после утверждения фонда: `conduct@example.org`.
