# Conference Application Template

## Title

Ocean Fund: Open Infrastructure for Ocean Research, Data, Education, and Public Engagement

## Format

- Talk
- Panel contribution
- Workshop
- Exhibition booth
- Poster

## Short Abstract

Фонд «Океан» формирует открытый публичный штаб для ocean-impact проекта: миссия, исследовательские направления, источники океанических данных, воспроизводимые notebooks, партнерские материалы и дорожная карта. Выступление показывает, как объединять научную аккуратность, открытые данные и общественное участие без неподтвержденных заявлений и закрытых материалов.

## Audience Fit

Подходит для аудиторий, связанных с океаном, климатом, биоразнообразием, научной коммуникацией, open data, museums, citizen science и международными партнерствами.

## Learning Outcomes

- понять, как структурировать открытый ocean-impact проект;
- увидеть карту источников данных: Copernicus Marine, OBIS, GEBCO, EMODnet, NOAA / IOOS, FathomNet;
- обсудить правила публичной проверки материалов;
- найти форматы участия для ученых, разработчиков и образовательных организаций.

## Requirements

- экран или проектор;
- возможность показать GitHub-репозиторий;
- для workshop: участники с ноутбуками и доступом к интернету.

## Notes

Перед подачей добавить конкретные данные события, лимит слов, биографию спикера и подтвержденные контакты фонда.
