# Целевые организации

Список помогает системно искать партнеров. Записи не означают существующее партнерство.

## Категории

| Категория | Что искать | Первый шаг |
| --- | --- | --- |
| Университеты | Морские науки, климат, экология, data science | Найти кафедры и публичные программы |
| Научные центры | Океанография, биоразнообразие, батиметрия | Изучить открытые проекты и контакты для партнерств |
| Музеи и научные центры | Океан, климат, планета, технологии | Предложить лекцию, workshop или визуализацию |
| Фонды | Океан, климат, образование, open science | Проверить грантовые направления |
| Конференции | Ocean science, climate, open data, museums | Отследить сроки заявок |
| Open-source сообщества | Data visualization, geospatial, ML, notebooks | Предложить конкретные data tasks |

## Карточка организации

| Поле | Описание |
| --- | --- |
| Название | Официальное название организации |
| Страна/регион | Где работает организация |
| Тип | University, museum, foundation, conference, NGO, lab |
| Тема | Почему организация релевантна фонду |
| Ссылка | Публичная ссылка |
| Возможный формат | Talk, workshop, data review, exhibition, grant, volunteer task |
| Статус | `watching`, `contact planned`, `contacted`, `discussion`, `closed` |

Не добавляйте личные email и телефоны в этот публичный файл.
