## Правила коммуникации

- не обещать партнерство до подтверждения;
- не публиковать персональные контакты;
- писать спокойно, конкретно и без рекламных преувеличений;
- указывать, что фонд находится в процессе развития открытой инфраструктуры;
- фиксировать результат контакта в issue или private CRM, если контакт содержит персональные данные.
