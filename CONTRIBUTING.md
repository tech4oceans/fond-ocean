# Участие / Contributing

Спасибо за интерес к фонду «Океан». Репозиторий открыт для аккуратных вкладов в исследования, данные, документацию, презентации и партнерскую коммуникацию.

## Как начать

1. Прочитайте [`README.md`](README.md), [`docs/mission.md`](docs/mission.md) и [`docs/governance.md`](docs/governance.md).
2. Выберите небольшую задачу или создайте issue по шаблону.
3. Делайте pull request с одним понятным изменением.
4. Указывайте источники, даты доступа и ограничения.

## Что особенно полезно

- карточки открытых источников данных;
- уточнение терминов в глоссарии;
- исследовательские вопросы;
- шаблоны для конференций и партнерств;
- проверка ссылок и лицензий;
- улучшение навигации и структуры.

## Правила безопасности

Не добавляйте:

- токены, пароли, API keys, webhook URL;
- персональные документы;
- личные email, телефоны и адреса;
- банковские реквизиты;
- неподтвержденные партнерства;
- закрытые данные из внутренних систем.

## Стиль

- основной язык сейчас русский;
- важные публичные заголовки можно делать RU/EN;
- тон спокойный, научный и международно понятный;
- без рекламных обещаний;
- все материалы относятся только к фонду «Океан».
