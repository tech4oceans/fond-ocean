# 海洋生物多样性

![ocean-biodiversity preview](../../assets/research-previews/zh/ocean-biodiversity.svg)


## 重点

研究海洋生物多样性有助于评估生态系统的健康状况，跟踪栖息地变化，找出观测中的差距，并教育社会了解海洋的价值。

## 研究问题

- 哪些开源来源提供了有关海洋物种出现的可验证数据？
- 跨区域、深度和分类群的观察差距在哪里？
- 哪些指标可用于教育和公共材料？
- 如何在不简化科学意义的情况下正确地可视化生物多样性？

## 潜在来源

| 来源 | 可能的应用 |
| --- | --- |
| 奥比斯 | 物种发生、分类记录、观测地理 |
| 深探网 | 带注释的水下图像和计算机视觉任务 |
| GBIF | 如果许可证和质量合适，则有关生物多样性的其他背景 |
| 科学出版物 | 检查方法、术语和限制 |

## 可能的结果

- 海洋生物多样性来源图；
- 公共材料指标清单；
- 带有加载打开记录示例的笔记本；
- 博物馆和教育场所的简短合作伙伴简介。

## 限制

物种出现数据可能不完整，并且因地区和观察方法而存在偏差。任何可视化都必须清楚地描述来源、访问日期和限制。
