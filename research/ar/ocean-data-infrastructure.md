# البنية التحتية لبيانات المحيطات

![ocean-data-infrastructure preview](../../assets/research-previews/ar/ocean-data-infrastructure.svg)


## ركز

البنية التحتية للبيانات هي أكثر من مجرد ملفات. هذه هي المصادر وبيانات التعريف والتراخيص وطرق الوصول والإصدارات والدفاتر والمرئيات وفحوصات الجودة وقواعد النشر.

## هدف

اجعل العمل مع بيانات المحيطات واضحًا للباحثين والمطورين والمتطوعين وشركاء المؤسسات.

## عناصر

| عنصر | لماذا هو مطلوب؟ |
| --- | --- |
| سجل المصادر | افهم بسرعة مكان الحصول على البيانات |
| بطاقات مجموعة البيانات | سجل الترخيص والتغطية والشكل والقيود |
| دفاتر الملاحظات | إظهار أمثلة التحليل القابلة للتكرار |
| البيانات الوصفية | حفظ السياق وتاريخ المراجعة |
| قواعد النشر | منع البيانات الخاصة والاستنتاجات غير المؤكدة |

## المهام الأولى

- املأ [`datasets-register.md`](../../data/datasets-register.md);
- حدد مصدرًا واحدًا مفتوحًا لدفتر الملاحظات التجريبي؛
- تحديد الحد الأدنى القياسي لبطاقة مجموعة البيانات؛
- وصف قواعد تخزين البيانات المشتقة.

## معايير الجودة

- المصدر متاح للعامة؛
- الرخصة واضحة؛
- تاريخ الوصول المشار إليه؛
- هناك وصف للقيود.
- يمكن تكرار التحليل.
